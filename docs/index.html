Snafu

A terse, dynamic, multi-paradigm programming language with ~165 features spanning 15+ paradigms.

Dynamic typing Pattern matching Algebraic effects APL-style arrays Code golf Self-modification Prototype OOP Concurrency

What is Snafu?

Snafu draws inspiration from Python (readability, dynamic typing), K and APL (array operators, terseness), Haskell (pattern matching, algebraic types), Lisp (code-is-data, eval), and INTERCAL (self-modification, comefrom). The interpreter is a single ~7000-line Python file.

Every keyword is 1-3 characters. Longer full-word aliases exist for discoverability, but the short form is canonical. Everything is modifiable at runtime: bindings, prototypes, methods, operators, the parser, even the interpreter itself.

Quick Start

$ python snafu.py
$ python snafu.py script.snf
$ python snafu.py -e "1 + 2"

No installation beyond Python 3 is required. Download snafu.py and run it directly. The first command opens a REPL, the second runs a file, and the third evaluates an expression.

A Taste of Snafu

FizzBuzz in one line
m(1..=15, `mt[a%3,a%5]{[0,0]->"FizzBuzz",[0,_]->"Fizz",[_,0]->"Buzz",_->a}`).each(P)
Sum types + pattern matching
sm Shape = Circle(r) | Rect(w, h)

df area(s) {
  mt s {
    Circle(r)  -> 3.14159 * r * r
    Rect(w, h) -> w * h
  }
}

area(Circle(5))            # 78.53975
Pipeline + APL array ops
[1,2,3,4,5] |> fl(`a%2==0`) |> m(`a*a`)   # [4, 16]

+/ [1,2,3,4]                      # 10  (reduce)
+\ [1,2,3,4]                      # [1, 3, 6, 10]  (scan)

Design Priorities

  1. Terseness. Canonical keywords are 1-3 characters. Longer aliases exist for discoverability.
  2. Dynamism. Everything is modifiable at runtime: bindings, prototypes, methods, operators, the parser, the interpreter.
  3. Meta-circularity. The interpreter is itself exposed and modifiable from within Snafu.
  4. Uniformity. Code is data. Algebraic effects subsume exceptions, generators, async, and reactive callbacks.
  5. Correctness over speed. The language spec chooses the semantically clean option, even when slow.
